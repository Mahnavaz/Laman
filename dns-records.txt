═══════════════════════════════════════════════════════════════
        تنظیمات DNS برای اتصال lamancity.com به Vercel
═══════════════════════════════════════════════════════════════

📍 مرحله 1: وارد پنل Parspack شوید
   https://my.parspack.co/hosting/client-area/domains/gYUcbvWQtD9/overview

📍 مرحله 2: به بخش "اطلاعات دامنه" بروید
   در تصویری که فرستادید، روی "اطلاعات دامنه" کلیک کنید

📍 مرحله 3: رکوردهای DNS را اضافه کنید

┌─────────────────────────────────────────────────────────────┐
│  رکورد 1: دامنه اصلی                                        │
├─────────────────────────────────────────────────────────────┤
│  Type:     A                                                │
│  Name:     @                                                │
│  Value:    76.76.21.21                                      │
│  TTL:      3600                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  رکورد 2: ساب‌دامین www                                     │
├─────────────────────────────────────────────────────────────┤
│  Type:     A                                                │
│  Name:     www                                              │
│  Value:    76.76.21.21                                      │
│  TTL:      3600                                             │
└─────────────────────────────────────────────────────────────┘

📍 مرحله 4: ذخیره کنید و منتظر بمانید
   - تغییرات DNS معمولاً 5-30 دقیقه طول می‌کشد
   - در برخی موارد تا 48 ساعت زمان می‌برد

📍 مرحله 5: بررسی کنید
   بعد از 10-15 دقیقه، سایت را باز کنید:
   ✓ http://lamancity.com
   ✓ http://www.lamancity.com

═══════════════════════════════════════════════════════════════

⚠️  نکات مهم:
   • رکوردهای قدیمی A را حذف کنید (اگر وجود دارد)
   • فقط IP آدرس 76.76.21.21 را وارد کنید
   • SSL (HTTPS) به‌صورت خودکار فعال می‌شود
